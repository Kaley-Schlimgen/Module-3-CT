<h1>C++ Program: Integer Pointers Program</h1>


Class: Programming III

Language: C++

Environment: XCode

<h2>Description</h2>
Program asks for the user to enter three integers and assigns them to three different variables, creates an integer pointer to dynamic memory for each variable, then displays the contents of the variables and pointers.

The program must include the new operator and the delete operator.
